# Swiggy Excel Dashboard

An interactive Excel-based business analytics dashboard designed to analyse Swiggy’s sales performance, customer ratings, and order trends across India.  
The dashboard transforms raw data into meaningful insights using pivot tables, charts, and slicers to support data-driven decision-making.

---

##  Key Results
- **Total Sales:** ₹53.01M  
- **Average Rating:** 4.34  
- **Ratings Count:** 5.59M  
- **Total Orders:** 197.43K  
- **Average Order Value:** ₹268.51  

---

##  Dashboard Insights
- **Monthly Sales Trend:** Sales fluctuations from January to August  
- **Daily & Weekly Trends:** Peak order days identified (Friday & Saturday)  
- **Food Type Sales:** Veg vs Non-Veg revenue comparison  
- **Top 5 Cities:** Highest revenue cities (e.g., Bangalore, Hyderabad)  
- **Sales by State:** India map visualising state-wise revenue  
- **Quarterly Summary:** Sales, ratings, and orders grouped by quarter  

---

##  Features
- Excel Pivot Tables & Pivot Charts  
- Interactive slicers (Month, Category, Restaurant)  
- KPI cards for quick performance overview  
- Clean and visually consistent dashboard layout
